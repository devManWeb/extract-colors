# extract-colors
A script to collect the colors of an image (maximum 20kB).

To work, you need to install [Pillow](https://pypi.org/project/Pillow/) on your system ([License](https://github.com/python-pillow/Pillow/blob/master/LICENSE))
```
 pip install Pillow
```

Please note, on Linux please check if you have tkinter installed.
